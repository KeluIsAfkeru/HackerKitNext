## ✨ 特性亮点

- Material You 风格，动画细腻，明暗模式无缝切换
- 代码架构优雅，UI/业务彻底解耦，易扩展
- 跨平台支持 Android / iOS / Web / Windows
- 智能状态持久化，使用体验极致流畅
- 定期维护，持续进化，追求极致

## 📦 快速开始

```bash
git clone https://github.com/KeluIsAfkeru/HackerKitNext.git
cd HackerKitNext
flutter pub get
flutter run
```
